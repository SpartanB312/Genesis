# Genesis

Java bytecode assembler and class-gen utils

## Assembler language support

* [X] Java
* [X] Kotlin
* [X] Scala

| Language   | Assembler type         | Author     |
|------------|------------------------|------------|
| **Java**   | Receiver & ChainCall   | B_312      |
| **Kotlin** | Kotlin DSL Receiver    | B_312      |
| **Scala**  | Contextual Abstraction | xiguajerry |
